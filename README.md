# Portfolio Template

A clean, responsive portfolio website template built with HTML, CSS, and JavaScript.

## Features

- **Hero** section with name and tagline
- **About** section for a personal bio
- **Skills** section displayed as tags
- **Projects** grid with card layout
- **Contact** form
- Responsive mobile navigation
- Smooth scrolling between sections

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in a browser.
3. Customise the placeholder text, colours, and links to make it your own.
